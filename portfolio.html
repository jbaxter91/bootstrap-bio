Iron Lotus

Iron lotus is a 2D side scrolling beat em up set in a world where steam punk fused with traditional japanese culture. This game had the most story built into it. There are 5 playable characters, all with their own backstory and purpose.

Project Murphy

Project Murphy started as a vague concept of turning a 3D game called "7 days to die" into a 2D top down game. This project really tested my ability to work with the unity lighting system as we really wanted to have a dark atmosphere and rely heavily on lights.

Blackjack Bot

A fun little side project I started to test my skills. This program was designed to play blackjack using basic strategy and a martingale betting pattern. Because this company gives you casino comps for playing, this bot ended up earning me a few free meals and free shows when i went to vegas on vacation.
